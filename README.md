GUI that relies on https://github.com/bastimeyer/dayz-linux-cli-launcher
It's made for me. You maybe you will find it useful.


You will need to add API key from battlemetrics either to the start script or your system enviromental variable.

Features: Searching DayZ servers by name, saving them to bookmarks, saving servers to bookmarks manually, bookmark system.
When combined with dayz-linux-cli-launcher acts as GUI launcher.
